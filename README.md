## How to use

switch to correct version of node

```bash
nvm use
```

install dependencies

```bash
yarn install
```

to spin up local dev server

```bash
yarn dev
```

to spin up local storybook server

```bash
yarn storybook
```
## Storybook ERR_REQUIRE_ESM
if you are encountering this error run
```bash
yarn upgrade storybook
```
